# Notação pós-fixa com ANTLR4
Converte uma expressão matemática simples no formato infixa para o formato pós-fixa, utilizando uma gramática do ANTLR4.  
Exemplo:<br>
<pre>
1+1 => 11+<br>
(2+3)/5 => 23+5/<br>
</pre>
